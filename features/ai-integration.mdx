## Overview

InsForge provides an OpenAI-compatible AI API powered by OpenRouter, giving you access to multiple AI providers and models through a single interface.

### Key Features

- **Multiple Providers** - OpenAI, Anthropic, Google, Meta, and more
- **Chat Completions** - Text generation with streaming support
- **Image Generation** - DALL-E, Stable Diffusion, and other models
- **Embeddings** - Vector embeddings for semantic search
- **Function Calling** - Tool use and structured outputs
- **Web Search** - AI with internet access
- **File Parser** - Process PDFs and documents
- **Extended Thinking** - Deep reasoning capabilities

## Architecture

```mermaid
graph LR
    A[Client] -->|Request| B[InsForge API]
    B -->|Proxy| C[OpenRouter]
    C -->|Route| D[AI Provider]
    D -->|OpenAI/Anthropic/etc| E[Model]
    E --> D
    D --> C
    C --> B
    B --> A
```

## Chat Completions

### Basic Chat

<CodeGroup>
```typescript TypeScript SDK
import { createClient } from '@insforge/sdk';

const client = createClient({
  baseUrl: 'https://your-app.region.insforge.app',
  anonKey: 'your-anon-key'
});

const { data, error } = await client.ai.chat.completions.create({
  model: 'openai/gpt-4',
  messages: [
    { role: 'user', content: 'What is InsForge?' }
  ]
});

if (error) {
  console.error('AI error:', error);
} else {
  console.log('Response:', data.text);
}
```

```bash cURL
curl -X POST https://your-app.region.insforge.app/api/ai/chat/completion \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "openai/gpt-4",
    "messages": [
      {"role": "user", "content": "What is InsForge?"}
    ]
  }'
```
</CodeGroup>

Response:
```json
{
  "text": "InsForge is a backend-as-a-service platform...",
  "metadata": {
    "model": "openai/gpt-4",
    "usage": {
      "promptTokens": 15,
      "completionTokens": 42,
      "totalTokens": 57
    }
  }
}
```

<CodeGroup>

const messages = [

```typescript React Chat Example
import { useState } from 'react';

function ChatBot() {
  const [messages, setMessages] = useState([
    { role: 'system', content: 'You are a helpful assistant.' }
  ]);
  const [input, setInput] = useState('');
  
  const sendMessage = async () => {
    const newMessages = [
      ...messages,
      { role: 'user', content: input }
    ];
    
    setMessages(newMessages);
    setInput('');
    
    const { data } = await client.ai.chat.completions.create({
      model: 'openai/gpt-4',
      messages: newMessages
    });
    
    setMessages([
      ...newMessages,
      { role: 'assistant', content: data.text }
    ]);
  };
  
  return (
    <div>
      {messages.map((msg, i) => (
        <div key={i}>{msg.role}: {msg.content}</div>
      ))}
      <input value={input} onChange={e => setInput(e.target.value)} />
      <button onClick={sendMessage}>Send</button>
    </div>
  );
}
```
</CodeGroup>

### Streaming Responses

Stream responses for real-time output:

<CodeGroup>
```typescript TypeScript SDK
const { data, error } = await client.ai.chat.completions.create({
  model: 'openai/gpt-4',
  messages: [
    { role: 'user', content: 'Write a short story.' }
  ],
  stream: true
});

if (!error) {
  for await (const chunk of data) {
    process.stdout.write(chunk);
  }
}
```

```typescript React Streaming
function StreamingChat() {
  const [response, setResponse] = useState('');
  
  const chat = async () => {
    setResponse('');
    
    const { data } = await client.ai.chat.completions.create({
      model: 'openai/gpt-4',
      messages: [{ role: 'user', content: 'Write a poem.' }],
      stream: true
    });
    
    for await (const chunk of data) {
      setResponse(prev => prev + chunk);
    }
  };
  
  return (
    <div>
      <button onClick={chat}>Generate Poem</button>
      <pre>{response}</pre>
    </div>
  );
}
```
</CodeGroup>

## Function Calling

Use AI to call structured functions:

<CodeGroup>
```typescript Define Tools
const tools = [
  {
    type: 'function',
    function: {
      name: 'get_weather',
      description: 'Get the current weather for a location',
      parameters: {
        type: 'object',
        properties: {
          location: {
            type: 'string',
            description: 'City name'
          },
          unit: {
            type: 'string',
            enum: ['celsius', 'fahrenheit']
          }
        },
        required: ['location']
      }
    }
  }
];
```

```typescript Call with Tools
const { data } = await client.ai.chat.completions.create({
  model: 'openai/gpt-4',
  messages: [
    { role: 'user', content: 'What is the weather in Paris?' }
  ],
  tools
});

if (data.tool_calls) {
  for (const call of data.tool_calls) {
    console.log('Function:', call.function.name);
    console.log('Arguments:', JSON.parse(call.function.arguments));
  }
}
```

```typescript Complete Flow
// 1. AI decides to call function
const { data: response1 } = await client.ai.chat.completions.create({
  model: 'openai/gpt-4',
  messages: [
    { role: 'user', content: 'Weather in Paris?' }
  ],
  tools
});

// 2. Execute function
const toolCall = response1.tool_calls[0];
const args = JSON.parse(toolCall.function.arguments);
const weather = await getWeather(args.location);

// 3. Send result back to AI
const { data: response2 } = await client.ai.chat.completions.create({
  model: 'openai/gpt-4',
  messages: [
    { role: 'user', content: 'Weather in Paris?' },
    { role: 'assistant', content: null, tool_calls: [toolCall] },
    { 
      role: 'tool',
      tool_call_id: toolCall.id,
      content: JSON.stringify(weather)
    }
  ],
  tools
});

console.log('Final response:', response2.text);
```
</CodeGroup>

## Web Search

Enable AI with internet access:

<CodeGroup>
```typescript Native Search
const { data } = await client.ai.chat.completions.create({
  model: 'openai/gpt-4',
  messages: [
    { role: 'user', content: 'What are the latest AI developments?' }
  ],
  webSearch: {
    enabled: true,
    engine: 'native' // Uses provider's built-in search
  }
});
```

```typescript With Citations
const { data } = await client.ai.chat.completions.create({
  model: 'openai/gpt-4',
  messages: [
    { role: 'user', content: 'Latest tech news?' }
  ],
  webSearch: { enabled: true }
});

console.log('Response:', data.text);

if (data.annotations) {
  console.log('Sources:');
  data.annotations.forEach(ann => {
    console.log(`- ${ann.urlCitation.title}: ${ann.urlCitation.url}`);
  });
}
```
</CodeGroup>

## Embeddings

Generate vector embeddings for semantic search:

<CodeGroup>
```typescript TypeScript SDK
const { data, error } = await client.ai.embeddings.create({
  model: 'openai/text-embedding-3-small',
  input: 'InsForge is a backend platform'
});

if (!error) {
  const embedding = data.data[0].embedding;
  console.log('Vector dimension:', embedding.length);
  
  // Store in database with pgvector
  await client.database
    .from('documents')
    .insert([{
      content: 'InsForge is a backend platform',
      embedding: embedding
    }]);
}
```

```typescript Batch Embeddings
const { data } = await client.ai.embeddings.create({
  model: 'openai/text-embedding-3-small',
  input: [
    'First document',
    'Second document',
    'Third document'
  ]
});

data.data.forEach((item, i) => {
  console.log(`Document ${i} embedding:`, item.embedding);
});
```
</CodeGroup>
